# Project Description

HELIX is a web-based Global Natural Disaster Monitoring & Analytics Platform that utilizes real-time disaster event data provided by the NASA EONET API.

The application is designed as an implementation of NoSQL Data Modeling Architecture using MongoDB Community Edition. The primary objective is not only to visualize disaster information but also to demonstrate proper NoSQL database design, including embedded documents, reference documents, denormalization, indexing, aggregation, and query optimization.

The system periodically retrieves disaster data from NASA EONET, transforms the raw response into a custom MongoDB schema, stores the processed documents, generates aggregation collections, and provides analytics through a modern web dashboard.

---

# Project Objectives

The objectives of HELIX are:

* Build a web-based global disaster monitoring dashboard.
* Integrate real-time data from NASA EONET API.
* Design an optimized MongoDB schema based on query-oriented modeling.
* Implement MongoDB indexing and aggregation.
* Demonstrate embedded document and reference document concepts.
* Visualize disaster information using an interactive dashboard.
